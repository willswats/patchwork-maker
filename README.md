# Patchwork Maker

A program for my programming coursework.

## Install `graphics.py` with `venv` (bash/zsh)

```bash
python3 -m venv venv
source venv/bin/activate
pip install graphics.py
```

## Install `graphics.py` with `venv` (fish)

```bash
python3 -m venv venv
source venv/bin/activate.fish
pip install graphics.py
```
